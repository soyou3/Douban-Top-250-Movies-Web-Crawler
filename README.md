# 豆瓣Top250电影爬虫与可视化分析工具

一个用于爬取豆瓣电影Top250榜单数据并进行可视化分析的Python工具，可生成电影类型分布、地区评分、年份趋势和演员出镜频次等分析图表。

## 功能特点

- 爬取豆瓣电影Top250数据（支持指定1-10页，每页25部电影）
- 对电影数据进行多维度分析：
  - 电影类型数量分布
  - 各地区电影平均分对比
  - 各年份电影数量趋势
  - 演员出镜频次Top15统计
- 自动生成可视化图表并保存为图片文件
- 支持加载本地已保存的数据进行分析

## 环境要求

- Python 3.7+
